# Cardiovascular Medical Device Market Analysis

## Overview
Market analysis of U.S. cardiovascular medical device trends using 
publicly available FDA 510(k) clearance data and CMS Medicare 
utilization reports (2015–2023).

## Objectives
- Map market entry trends across cardiovascular device categories
- Identify top manufacturers by clearance volume
- Forecast procedure volume growth using time-series analysis
- Build a competitive landscape from regulatory filing patterns

## Data Sources
- FDA 510(k) Premarket Notification Database (publicly available)
- CMS Medicare Physician & Supplier Procedure Summary
- Source links included in /data/sources.md

## Tools Used
- Python (Pandas, Matplotlib, statsmodels)
- Excel (pivot tables, advanced formulas)
- Tableau / Power BI for dashboards
